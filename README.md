# Quick Start

---

## Install Dependencies

`composer install`

## Run Migrations

`php artisan migrate`

## Import Articles

`php artisan db:seed`

## If you get an error about an encryption key

`php artisan key:generate`

## Install JS Dependencies

`npm install`

## Watch Files

`npm run watch`
